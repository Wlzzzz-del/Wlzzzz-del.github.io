# 终端下设置代理
```
export http_proxy=http://127.0.0.1:8889
```
# git设置代理
```
git config --global http.proxy http://127.0.0.1:8889
git config --global --unset http.proxy
```
# npm设置代理
```
npm config set proxy=http://127.0.0.1:8889
```
# npm取消代理
```
npm config delete proxy
```
